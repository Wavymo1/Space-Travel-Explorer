# Space Travel Explorer

## Brief Description
Space Travel Explorer is a CS430 multimedia website where the user chooses a planet, launches through a full-screen space travel sequence, and lands on a dashboard with a rotating planet, mission-style facts, audio, and interactive features.

## Team Members and Roles
- Mohamed Allsayed — completed the working site, including the page structure, layout, planet selector, warp animation, rotating planet display, quiz, contact form validation, source page, README, and final report.
- Other group members — no specific files, code, or written sections were available to include in this submitted version.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Canvas API
- Local audio
- Local image textures
- SVG
- GitHub Pages

## Key Features
- Multi-page navigation: Explorer, About Mission, Join Mission, and Sources
- Full-screen canvas travel animation
- Rotating planet visuals with local texture files
- Saturn ring display with separate front and back ring layers
- NASA fact links for every planet
- Local cockpit ambience audio
- Interactive quiz
- Mission signup form with client-side validation
- Responsive layout for desktop and mobile
- Accessibility features such as semantic tags, skip link, readable contrast, focus states, labels, and reduced-motion support
